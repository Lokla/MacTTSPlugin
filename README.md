# MacTTSPlugin

A low-latency ACT plugin for macOS/CrossOver users.

Project sources are in MacTTSPlugin.

See build and install docs in MacTTSPlugin/README.md.

Note: `.github/workflows/build.yml` artifacts are compile-check outputs built
against a stub ACT API. Release workflow artifacts are built in CI against the
official ACT download as an external compile-time reference.

## Attribution

This project was inspired by and uses the LinuxTTSPlugin project as a blueprint
for ACT plugin structure and workflow patterns:

- LinuxTTSPlugin: https://github.com/Minizbot2012/LinuxTTSPlugin

MacTTSPlugin is a separate macOS-focused implementation.
